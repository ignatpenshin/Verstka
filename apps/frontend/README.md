# Verstka Frontend

React + TipTap + Yjs collaborative editor frontend.

## Features

- Real-time collaborative editing with CRDT (Yjs)
- Rich Markdown editor (TipTap)
- Math rendering (KaTeX)
- Diagram support (Mermaid)
- Live preview
- File tree browser
- Comments and review mode

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
